# Armazenando dados no navegador

</br>

Não esqueça mais de colocar seus itens na mochila de viagem!


<img alt="Imagem de uma mochila com uma lista de itens ao lado direito" width="763" src="https://github.com/anadezuo/mochila-de-viagem/assets/43011663/37f7d581-0303-4e26-bbe9-f153aa78be21">


Com esse projeto é possível acrescentar um item na lista, remover, adicionar ou atualizar a quantidade do item.

E ainda pode adicionar uma observação sobre onde está o item, ou se ainda precisa correr no shopping.

</br>
</br>

⚙️ Funcionalidades

✅ Inserir novos itens

✅ Inserir descrições

✅ Adicionar ou substituir o item

✅ Adicionar uma lista de intes pré definidos

</br>
</br>

⏲️ Funcionalidades futuras

Permitir alterar o item na linha

Personalizar sua lista de itens default
